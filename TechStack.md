Frontend: React, Tailwind CSS (for custom utility styling alongside Stitch components)

UI/Components: Stitch MCP (via google-labs-code/stitch-skills)

Backend: Node.js with Express.js

Database: MongoDB (to store contact form submissions)

Security & Middleware: cors (restrict API access), helmet (secure HTTP headers), express-rate-limit (prevent spam), express-mongo-sanitize (prevent injection).

Deployment: Vercel (Frontend) and Render or Railway (Backend)
